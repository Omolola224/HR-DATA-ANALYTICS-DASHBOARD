# HR DATA ANALYTICS-PowerBI
## Project Overview
This project analyzes Employee/HR data using PowerBI to uncover ratings insights based on gender, identify pay gaps, and employee-related metrics across departments and regions for better decision making.
## Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel
- DAX
- Data Cleaning
- Data Visualization
## Key KPIs
- Total Employees
- Average Salary
- Total Bonus
- Average Performance Rating
## Dashboard Features
### Employee Analysis
- Employee Distribution by Department
- Gender Representation Analysis
### Salary Analysis
- Salary Distribution
- Regional Salary Comparison
- Salary Bands
### Performance Analysis
- Performance rating comparison
### Bonus Analysis
- Regional bonus payouts
- Bonus allocation patterns
## Key Insights
- Some departments had noticeable gender imbalance
- Salary distribution varied across regions and departments
- A few employees earned below the $90,000 minimum requirement
- Bonus allocation differed across regions
- Performance ratings were relatively balanced across genders
## Recommendations
- Improve gender diversity in under-represented departments
- Review salary structure to address possible pay gaps
- Adjust salaries below the required minimum threshold
- Standardize bonus allcation across regions
- Continue using HR Analytics for workforce decision-making
